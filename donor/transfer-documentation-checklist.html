Quick tips

  • Store a signed deed and box list together in a single folder / PDF for handoff.
  • Mark restrictions precisely (series and box numbers) and include review dates.
  • Digitization documentation should include file formats, checksums, and folder structure.
  • Keep a contact name, email, and phone on the first page of the box list.
